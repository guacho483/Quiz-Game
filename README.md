# Quiz-Game


Quiz game è un gioco che simula un quiz con varie domande(10 in totale) fornite da un database di domande online, ossia openTrivia DB.Il programma instaura una connessione e ricava la domanda da porre tramite la GUI con una fetch all'URL del sito openTrivia.
Nel programma è stata implementata una interfaccia, come richiesto, a schede, dove la prima è una scheda di benvenuto,la seconda riguarda il quiz effettivo, e la terza il punteggio totale ottenuto.
